# 책 읽고 정리하는 곳

- 책을 정리하면서 읽는 이유
  - 눈으로만 읽으면 잠깐 딴생각이 들때 눈으론 읽었지만 머리엔 남지 않는 경우가 있음
  - 정리를 하면 읽은 부분을 정리하기 때문에 모든 페이지를 커버할 수 있음
  - 책을 읽고 시간이 지나면 잊어버리지만 내가 정리했던 것을 보면 기억나게 할 수 있음
  - 출퇴근 시간에 지하철에서 복습할 수 있음
  - 느리지만 확실한 방법
