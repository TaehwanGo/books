#### 표 15-1. 분산 캐시 대 복제 캐시

| 결정 기준     | 복제 캐시     | 분산 캐시    |
| ------------- | ------------- | ------------ |
| 최적화        | 성능          | 일관성       |
| 캐시 크기     | 작다(<100MB)  | 크다(>500MB) |
| 데이터 유형   | 비교적 정적임 | 매우 동적임  |
| 업데이트 빈도 | 비교적 낮음   | 매우 높음    |
| 내고장성      | 좋음          | 나쁨         |
